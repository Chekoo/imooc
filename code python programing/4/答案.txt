4.2
不可更改：对象属性的值不能更改
不可改变：元组、字符串、数字

4.3
顺序访问：list, truple, string
与映射不同的时，映射需要索引来访问，它的索引并不使用顺序的数字偏移量取值， 它的元素
无序存放，通过一个唯一的 key 来访问

4.4
type() 函数是取得对象类型的函数。type()返回的对象是对象的类型

4.5
不同之处：str()输出时候创建了一个新的对象，从而对人来说，可读性更好。repr()不同，得到字符串
可以重新获得该对象，并且能应用eval()函数   repr()等价于``运算符

4.6
type(a)==type(b)能通过再次调用type函数来对比a,b的类型值是否相同，而type(a) is type(b)是根据
对象类型属性进行对比，前者是bool运算，后者是对象类型比较。
函数isinstance()也是通过对象类型进行判断。


4.8
列表和元组都是序列访问，属于存储模型的，不同点是元组是不可更改对象

4.9
整型对象和字符串对象是不可变对象，所以Python会很高效地缓存它们。
